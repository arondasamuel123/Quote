# Description
This is  an app that creates quotes,deletes quotes, highlights quotes with the most upvotes and shows how long since a quote has been posted

# Author
Samuel Aronda

# Setup/Installation
1. Clone repository
2. Open folder and npm install 
3. Run ng serve on the terminal
